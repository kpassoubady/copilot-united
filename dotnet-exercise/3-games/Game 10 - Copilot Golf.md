## The Challenge

## The Setup

Students need to implement a fixed-size cache that removes the least recently accessed item when full.

**Hint**: Consider using `LinkedList<T>` combined with `Dictionary<TKey, LinkedListNode<T>>` or the built-in `MemoryCache` class.
