# GitHub Copilot Integration with VS Code and IntelliJ (December 2025)

## VS Code Setup

### 1. Install Extensions

Install both extensions from the VS Code Marketplace:
- **GitHub Copilot** — Core code completion
- **GitHub Copilot Chat** — Chat interface (included in Copilot extension as of 2024)

```
Extensions View: Ctrl+Shift+X (Windows/Linux) | Cmd+Shift+X (Mac)
Search: "GitHub Copilot"
```

### 2. Sign In to GitHub

1. After installation, click the Copilot icon in the status bar
2. Select "Sign in to GitHub"
3. Complete browser authentication
4. Verify subscription is active (Individual, Business, or Enterprise)

### 3. Configure Settings

Open Settings (`Ctrl+,` / `Cmd+,`) and search for "Copilot":

| Setting | Description |
|---------|-------------|
| `github.copilot.enable` | Enable/disable for specific languages |
| `github.copilot.advanced.model` | Select default model |
| `github.copilot.chat.localeOverride` | Language preference for responses |

### 4. Key Features in VS Code

- **Inline Suggestions**: Type and see ghost text completions
- **Inline Chat**: `Cmd+I` / `Ctrl+I` for quick questions
- **Chat Panel**: Full chat interface in sidebar
- **Copilot Edits**: Multi-file editing (Command Palette → "Copilot Edits")
- **Model Picker**: Select GPT-4o, Claude 3.5, Gemini in chat

## IntelliJ IDEA Setup

### 1. Install Plugin

1. Open IntelliJ IDEA
2. Go to **Settings/Preferences → Plugins**
3. Search "GitHub Copilot" in Marketplace
4. Install and restart IDE

### 2. Sign In to GitHub

1. After restart, navigate to **Settings → GitHub Copilot**
2. Click "Sign in to GitHub"
3. Complete browser authentication

### 3. Configure Settings

**Settings → GitHub Copilot:**

| Setting | Description |
|---------|-------------|
| Enable completions | Toggle inline suggestions |
| Show completions automatically | Auto-show vs manual trigger |
| Model selection | Choose AI model for responses |

### 4. Key Features in IntelliJ

- **Inline Suggestions**: Ghost text as you type
- **Copilot Chat**: Tool window for conversations
- **Inline Chat**: Context menu → "Ask Copilot"
- **Model Selection**: Available in chat settings

## Troubleshooting

| Issue | Solution |
|-------|----------|
| No suggestions appearing | Check subscription status, verify sign-in |
| Chat not responding | Update extension/plugin to latest version |
| Model not available | Verify subscription tier supports selected model |
| Slow suggestions | Check network connection, try different model |

For detailed troubleshooting, see [VS-Code-GitHub-Copilot-Logs.md](VS-Code-GitHub-Copilot-Logs.md).
