# GitHub Copilot integration with VS Code and IntelliJ

## Integrating GitHub Copilot with VS Code

1. **Install the GitHub Copilot Extension**:
    - Open VS Code.
    - Go to the Extensions view (Ctrl+Shift+X or Cmd+Shift+X on Mac).
    - Search for "GitHub Copilot" and install the official extension by GitHub.

2. **Sign In to GitHub**:
    - After installation, VS Code will prompt you to sign in to your GitHub account.
    - Follow the authentication steps to link your account.

3. **Start Using Copilot**:
    - Open a code file.
    - Begin typing; Copilot will suggest code completions.
    - Accept suggestions with Tab or continue typing to refine.

## Integrating GitHub Copilot with IntelliJ IDEA

1. **Install the GitHub Copilot Plugin**:
    - Open IntelliJ IDEA.
    - Go to Settings/Preferences > Plugins.
    - Search for "GitHub Copilot" and install the plugin.

2. **Sign In to GitHub**:
    - After installation, IntelliJ will prompt you to sign in to your GitHub account.
    - Authenticate via the provided link.

3. **Start Using Copilot**:
    - Open a project and code file.
    - Type code; Copilot will provide inline suggestions.
    - Accept with Tab or use the suggestion menu.

For both IDEs, ensure you have a GitHub account with Copilot access. Refer to GitHub's documentation for troubleshooting.
