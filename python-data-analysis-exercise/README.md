## Learning Path

### Session 1: Setup & Introduction (1-setup/)
- **intro_pandas**: Introduction to Pandas library fundamentals
- **FAQs**: Common questions and troubleshooting

### Session 2: Hands-on Exercises (2-breakout/)
- **ExercisePandasWithCars**: Practical exercise analyzing car dataset
  - Data loading and inspection
  - Data cleaning and preprocessing
  - Exploratory Data Analysis (EDA)
  - Visualization with Matplotlib/Seaborn

### Session 3: Interactive Games (3-games/)
- **Data Manipulation**: Games 1, 4, 7, 8 - Pandas operations, cleaning, aggregations
- **Visualization**: Games 2, 5, 6 - Creating and reverse-engineering charts
- **Analysis & ML**: Games 3, 9, 10 - EDA, time series, ML pipelines

## Prerequisites

Before starting the exercises:

1. Complete the installation guide: [macOS](../project4/install/proj4-install-mac.md) | [Windows](../project4/install/proj4-install-win.md)
2. Have Python 3.9+ installed
3. Install required packages: `pip install -r ../project4/data-pipeline/requirements.txt`
4. Have GitHub Copilot enabled in your IDE

## Using GitHub Copilot

Throughout these exercises, practice using Copilot for:
- Writing data loading functions
- Creating data cleaning pipelines
- Generating visualization code
- Building analysis reports
